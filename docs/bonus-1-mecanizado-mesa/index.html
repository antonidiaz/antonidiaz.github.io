Bonus 2 Mecanizado Extra

Documentación Bonus 2.

Descripción Encargo

Generar las trayectorias para que la máquina pueda fabricar el diseño utilizando Fusion 360 en el espacio de trabajo MANUFACTURE. El recurso utilizado fue descargado de la página obrary.com

Diseño Utilizado

El siguiente modelo es una silla diseñada por Dylan Stallard. Este es un diseño open source de la web Obrary.com Esta silla llamada “The George” obtuvo el primer lugar en 2014 en “Chair Design Competition” que organizó Obrary con San Diego State University.

https://obrary.com/collections/designs-for-the-cnc-router/products/the-george

Aprovechamiento del Espacio

En esta ocasión las piezas a mecanizar están organizadas de manera que se maximice el material que se usará, utilizando todos los espacios disponibles, sin orden o jerarquía alguna, para poner partes de la pieza final que quiero lograr.

Es necesario dejar como minimo una distancia equivalente al 50% del diametro de la fresa (3mm en este caso) entre pieza y pieza para que no exista problemas en el proceso.

Proceso

1. Extruir las piezas

El archivo estaba en formato .3dm. En Rhinoceros extruir las piezas del tamaño que quería que tuviera el armado. En este caso 15 mm.

Luego, exporté las piezas en formato .STEP a Fusion 360 para hacer el mezanizado.

2. Contruir Stock

El stock para este diseño tiene la misma dimensión en el eje z para poder realizar un mecanizado simple solo en 2 dimensiones. Para crearlo dibujé un cuadrado (Sketch) y lo extruí del mismo espesor que las piezas (15 mm) en el espacio de trabajo DESIGN.

Luego, en el espacio de trabajo MANUFACTURE hacer el Set Up del espacio de trabajo (igual que en los mecanizados anteriores).

3. 2D Contour

Las piezas de este modelo no tiene relieves, por lo que la herramienta 2D Contour es suficiente para mecanizar todas las piezas.

Para simplicar el proceso de Set Up, agrupé las piezas similares en un ajusto diferente. A los cortonos les agregué TABS de 3mm con la distancia adecuada para el tamaño de cada una de las piezas.
