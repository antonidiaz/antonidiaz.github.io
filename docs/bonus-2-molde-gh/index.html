Bonus 1 Molde GH

Documentación del Bonus 1

Descripción Encargo

En este encargo se pide modelar en Rhinoceros una palmeta utilizando el código en bloques de grasshopper entregado para luego mecanizarlo en el espacio de trabajo MANUFACTURE en Fusion 360 con las técnicas aprendidas en el mecanizado de la silla.

Modelo en Fusion360

Vistas del Modelo

Pasos a Seguir

1. Crear Molde en Rhinoceros

  • Primero, abrir el código en bloque en Grasshopper y encenderlo para visualizar el diseño.

  • Crear superficie que quiero sustraer de la caja contenedora. En mi caso, hice tres esferas unidas repetidas 3 veces por lado.

  • Insertar superficie creada al código

      Brep > click derecho > Set one Brep > seleccionar figura creada

  • “Cocinar” el modelo para poder crear la superficie

      SDiff > click con la rueda del mouse > presionar el huevo frito (bake)

  • Por último, extraer el molde diseñado en formato .STEP

2. Crear el stock de mecanizado

Despues de importar la figura traida desde Rhinoceros a Fusion 360 y alinearla con el origen y crear la plancha de mecanizado. No le puse una medida especial sino que me preocupé de que sobrepasara la figura en todas las dimensiones.

El procedimiento del mecanizado sigue los mismos pasos que los realizados con el respaldo anteriormente.

3. 2D Contour

En este paso se realiza el contorno del molde con la herramienta 2D Contour con una fresa plana de 6mm agregando TABS y Multiple Depths con distancia de 3mm

3. Adaptive Clearing

En este paso se limpia y remueve material del eje Z para en un siguiente paso hacer las terminaciones. Para esto usamos la herramienta Adaptive Clearing entre las herramientas de 3D.

Al igual como lo hicimos en el respaldo, primero se debe preparar la figura proyectando el espacio de mecanizado en el area de trabajo DESIGN. Una vez hecho esto se realiza este set up utilizando la misma fresa de punta plana usada en el paso anterior.

Notar que el tamaño de la palmeta debe ser de un porte razonable para que el software pueda crear la trayectorias.

4. Parallel

En este último paso se realizan las teminaciones para que el modelo quede más prolijo. Para esto se utiliza Parallel ubicada entre las herramientas 3D. Utilizamos una fresa de las mismas dimensiones que para el paso anterior pero en este caso tiene que ser con punta redonda para conseguir un terminado más perfecto.

No olvidar realizar las trayectorias paralelas hacia ambos sentidos (perpendicular)
